# Korea_Base_Interest_Rate_Trend.csv

## 📄 파일 설명  
이 데이터셋은 **1999년부터 2025년 5월 9일까지**의 **대한민국 기준금리(정책금리)** 변동 내역을 포함하고 있습니다.  
해당 금리는 **한국은행**에서 발표한 공식 금리이며, 거시경제 분석, 금융 모델링, 금리 추세 예측 등에 활용할 수 있습니다.
